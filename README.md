# Gaia Agent HuggingFace

This repository contains a Jupyter Notebook (`gaia_agent_huggingfacet.ipynb`) demonstrating the use of Hugging Face agents and related tools.

## Prerequisites

The notebook requires the following Python packages:
- `smolagents`
- `datasets`
- `duckduckgo-search`
- `pandas`
- `pillow`
- `litellm`
- `tqdm`

## Usage

1. Clone the repository.
2. Open `gaia_agent_huggingfacet.ipynb` in Jupyter Notebook or Google Colab.
3. Run the cells to install dependencies and authenticate with Hugging Face using your token.
4. Follow the instructions within the notebook to interact with the agent.
